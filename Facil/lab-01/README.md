# WarLabs - Lab 001 SSH

### Nivel

**Fácil**
### Descripción

Laboratorio diseñado para practicar técnicas **básicas de enumeración** de puertos y **enumeración** web que permite la lectura de datos sensibles así como la obtención de usuarios.

Una vez obtenido el acceso al sistema, el laboratorio introduce una **capability** en el binario de python3, lo que permite una escalada de privilegios a root. 

### Objetivo

Obtener acceso al sistema mediante **fuerza bruta sobre SSH** y posteriormente escalar privilegios a root.

### Aprendizajes

- Enumeración web inicial.
- Identificación de información sensible.
- Filtración de nombres de usuario.
- Fuerza bruta **dirigida** contra el servicio **SSH**.
- Enumeración de capabilities en Linux
- Introducción a la escalada de privilegios.
